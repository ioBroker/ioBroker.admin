# NexoWatt EOS Admin

NexoWatt EOS Admin ist die gebrandete Administrationsoberfläche für die EOS-Basis.  
Der Fokus liegt auf einer konsistenten Oberfläche im Stil des NexoWatt UI Cockpits: dunkles Energie-Dashboard-Design, grüne Live-Akzente, klare Kartenoptik und EOS-Branding.

## Enthalten

- NexoWatt EOS Branding für Login, Topbar, Seitenmenü, Manifest und Adapter-Metadaten
- Dunkles Cockpit-Theme mit Glow-, Card- und Linienoptik passend zur NexoWatt UI
- neue EOS-Logos, Favicons und Energie-Icons
- Runtime-Branding-Guard, damit sichtbare Alttexte, Titel, Platzhalter und Übersetzungen auf NexoWatt EOS umgeschrieben werden
- upstream-nahe Struktur, damit spätere Basis-Updates möglichst konfliktarm übernommen werden können

## Update-Strategie

Die technische Adapter-Basis bleibt absichtlich nah am Original. Für zukünftige Updates wird empfohlen, den aktuellen upstream-Stand in einen NexoWatt-Branch zu rebasen und anschließend den Branding-Patch erneut anzuwenden.

Details stehen in `docs/NEXOWATT_EOS_BRANDING_DE.md`.

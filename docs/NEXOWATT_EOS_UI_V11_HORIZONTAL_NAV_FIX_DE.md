# NexoWatt EOS Admin UI v11

Diese Version setzt den App-Header über die komplette Breite, wandelt die linke Navigation in eine horizontale Navigationsleiste unterhalb des Headers um und lässt den Hauptbereich darunter über die komplette Seitenbreite laufen.

Korrekturen:

- Login-Karte wird vertikal und horizontal vermittelt.
- Abmelden-Button wird auf der Anmeldeseite ausgeblendet.
- NexoWatt-Logo wird über robuste absolute Asset-Pfade geladen, damit keine 404-Fehler durch relative `/login/...` Pfade entstehen.
- Adapter-Beschreibungen nach Drei-Punkte-Klick sind im dunklen EOS-Panel lesbar.
- Dienste-/Instanzenliste bleibt im EOS-Stil ohne NexoWatt-Logo auf einzelnen Instanzen.

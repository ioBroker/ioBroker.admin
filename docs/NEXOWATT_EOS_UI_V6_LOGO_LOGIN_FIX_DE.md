# NexoWatt EOS Admin UI v6

Änderungen:

- Login zeigt keinen Abmelden-Button mehr.
- Temporäres Kompass-Icon wurde entfernt. Es wird überall das ursprüngliche NexoWatt-Logo verwendet.
- Admin-/Adapter-Icons, PWA-Icons und Favicons wurden auf das NexoWatt-Logo vereinheitlicht.
- Runtime-Branding beobachtet keine Attribute mehr und reduziert damit DOM-Arbeit auf großen Modul-/Adapterseiten.
- Logout-URL bleibt relativ zur Admin-Basis, um 404-Probleme hinter Pfaden/Proxy-Setups zu vermeiden.
- Die technischen Admin-Bundles bleiben unangetastet.

# NexoWatt EOS UI v13

Diese Version behält das akzeptierte v12-Layout bei und ändert gezielt zwei Bereiche:

- Der zusätzliche EOS-Abmeldebutton wurde entfernt, weil er je nach Session-/Proxy-Pfad zu falschen Redirects führen konnte.
- Die System-/Basiseinstellungen wurden optisch an EOS angepasst: dunkle Feldkarten, bessere Labels, lesbare Tabs, dunkle Action-Bar und eingerahmte Karte.

Die upstream React/Vite Bundles bleiben unangetastet. Das Branding läuft weiterhin über `adminWww/css/eos-branding.css` und `adminWww/js/eos-branding.js`.

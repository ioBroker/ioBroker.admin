# NexoWatt EOS UI v9

Diese Version basiert wieder auf dem bevorzugten v5/v8-Shell-Design und enthält gezielte Feinschliffe:

- Das Logo oben links ist größer und sitzt mittig in der EOS-Branding-Kachel neben der Schrift.
- Die Adapter-/Modulbeschreibung nach Klick auf die drei Punkte ist dunkel, kontrastreich und lesbar.
- Die Dienste-/Instanzenliste erhält eine passendere EOS-Optik mit abgerundeten Zeilen, dunklen Panels und Akzentlinien.
- NexoWatt-Logos werden weiterhin nur auf Branding-Flächen verwendet und nicht auf einzelne Instanzen gestempelt.
- Cache-Busting wurde auf `v=9` gesetzt.
